Useless parentheses around expressions should be removed to prevent any misunderstanding (squid:UselessParenthesesCheck)

Useless parentheses can sometimes be misleading and so should be removed.

Noncompliant Code Example

return 3;
return (x);           // Noncompliant
return (x + 1);       // Noncompliant
int x = (y / 2 + 1);  // Noncompliant

Compliant Solution

return 3;
return x;
return x + 1;
int x = y / 2 + 1;
